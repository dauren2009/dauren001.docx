# Hotel Booking System Database Model

This project contains a physical data model for a hotel booking system.

The database model includes the following tables:
- Hotels
- RoomTypes
- Rooms
- Customers
- Bookings
- Payments
- Staff
- Services
- BookingServices
- CheckInOut

The model is designed in the 3rd Normal Form (3NF) and includes a many-to-many relationship between Bookings and Services implemented through the BookingServices table.
